# Smart Envelope Pro ✉️
한글(HWP) 오토메이션을 이용한 서류 봉투 주소 자동 입력 프로그램입니다.

## 주요 기능
- 도로명 주소 검색 및 우편번호 자동 입력
- 연락처 하이픈(-) 자동 변환
- 네이버/카카오 지도 위치 확인 연동
- 한글(HWP) 누름틀 기반 데이터 자동 기입

## 설치 및 실행 방법
1. Python 설치
2. 필수 라이브러리 설치: `pip install customtkinter pywin32 requests geopy`
3. 행정안전부 주소 API 키 발급 후 코드 내 입력
4. `envelope_bot.py` 실행
